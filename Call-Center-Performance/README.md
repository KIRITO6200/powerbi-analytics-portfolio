# 📞 Call Center Performance Dashboard

## 🎯 Objective
Provide real-time and analytical visibility on call center activity to support:
- SLA monitoring
- Abandonment control
- Staffing optimization
- Performance comparison by group and region

## 📊 Key KPIs
- Calls Presented
- Calls Treated
- Calls Abandoned
- Treatment Rate
- Abandonment Rate
- QS (Quality Score)
- Average Handling Time (DMT)
- Number of Active Agents
